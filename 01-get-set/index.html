01-get-set

Step 1: localStorage

  1. Open your Chrome Developer Tools
  2. Navigate to your JavaScript console
  3. type "localStorage" and press enter
  4. This is the localStorage object! The data exists even when you close the browser and reopen it

Step 2: localStorage.setItem

You can set items in localStorage using "localStorage.setItem()"

  1. Type "localStorage.setItem('secret', 1234)" and press enter
  2. Type "localStorage.setItem('name', 'danny')" and press enter
  3. Type "localStorage.setItem('age', 26)" and press enter
  4. type "localStorage" and press enter
  5. You should see all the items you just set! Do you see anything special about the numbers you inserted?

Step 3: localStorage.getItem

you can get items in localStorage using "localStorage.getItem()"

  1. Type "localStorage.getItem('secret')" and press enter
  2. Type "localStorage.getItem('name')" and press enter
  3. Type "localStorage.getItem('age')" and press enter
  4. You should see all each item you just set!

Congrats! Go to the next step!
